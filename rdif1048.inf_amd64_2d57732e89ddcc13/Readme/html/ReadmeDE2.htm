Treiber für Windows 10

Der Windows® 10-Treiber für Roland RD wird automatisch aus dem Internet auf Ihren Computer heruntergeladen, wenn das Roland RD mit dem Computer verbunden ist.
*Bei der Verwendung eines Touch-fähigen PCs ersetzen Sie die Ausdrücke „klicken“ und „mit der rechten Maustaste klicken“ in diesem Dokument entsprechend mit „tippen“ und „drücken und halten“.

Systemanforderungen

Betriebssystem
Microsoft® Windows® 10 (64-Bit-Edition/32-Bit-Edition)
*Windows 10 Mobile wird nicht unterstützt.
Computer
Windows-kompatibler Computer mit USB-Anschluss
*Dieses Produkt kann nicht mit einem USB 3.0-Anschluss verwendet werden, der nicht kompatibel mit den USB 2.0-Spezifikationen ist.
*Wenn dieses Produkt beim Anschließen an einen USB 3.0-Anschluss nicht ordnungsgemäß funktioniert, schließen Sie es an einen USB 2.0-Anschluss an.
*Die Leistung des Produkts ändert sich auch bei einem Anschluss an einen USB 3.0-Anschluss nicht.
*Dieser Treiber kann nicht in einer virtuellen Windows-Umgebung wie z. B. Hyper-V oder Virtual PC verwendet werden.
*Mac-Computer mit Microsoft Windows werden nicht unterstützt.
Weitere
Internetverbindung
*Zum Herunterladen des Treibers wird eine Internetverbindung benötigt.

Warnungen und Einschränkungen

  • Wenn Leistung und Spezifikationen Ihres Computers in irgendeiner Form unangemessen sind, können Tonunterbrechungen oder andere Unvollkommenheiten auftreten. Die von Ihnen genutzten Anwendungen und die Art ihrer Verwendung, können sich auf Ihre Erfahrung auswirken.
  • Wenn der Computer gestartet wird, während das Roland RD noch im eingeschalteten Zustand mit dem Computer verbunden ist, wird der Treiber möglicherweise nicht richtig geladen, und Sie können das Roland RD möglicherweise nicht verwenden. Schließen Sie in diesem Fall das Roland RD an einen anderen USB-Anschluss an, oder schalten Sie dasRoland RD nach dem Starten des Computers ein.
  • Wenn das Roland RD nach der Rückkehr aus dem Energie sparen-Modus nicht korrekt funktioniert, sollten Sie alle Anwendungen beenden, die das Roland RD verwenden, und anschließend das Roland RD aus- und wieder einschalten.
  • Wenn das USB-Kabel während der Verwendung des Roland RD entfernt und wieder angeschlossen wird oder wenn das Roland RD aus- und wieder eingeschaltet wird, kann es ggf. nicht mehr korrekt funktionieren. Nachdem der Computer aus dem Ruhezustand zurückgekehrt ist, sollten Sie in diesem Fall alle Anwendungen beenden, die das Roland RD verwenden, und das Roland RD aus- und wieder einschalten.
  • Wenn der Prozessor Ihres Computers bei Verwendung des Roland RD stark ausgelastet war (beispielsweise durch einen Zugriff auf ein CD-ROM-Laufwerk oder ein Netzwerk ), kann das Roland RD möglicherweise nicht mehr fehlerfrei betrieben werden. Halten Sie in diesem Fall die Wiedergabe oder Aufnahme an und versuchen Sie es erneut. Ist die Wiedergabe oder Aufnahme immer noch nicht möglich, beenden Sie alle Programme, die das Roland RD verwenden, und schalten Sie anschließend das Roland RD aus und wieder ein.

Installation

*Hinweise zur Neuinstallation oder Aktualisierung des Treibers finden Sie unter "Eine Neuinstallation vornehmen."
  1. Trennen Sie alle USB-Geräte außer Tastatur und Maus, und starten Sie dann Windows.
  2. Stellen Sie sicher, dass der Computer mit dem Internet verbunden ist.
  3. Schließen Sie unter Windows in der Systemsteuerung den Bildschirm mit den Einstellungen für „Sound“ und alle aktiven Anwendungen.
  4. Schalten Sie das Roland RD ein, und schließen Sie das Gerät mit einem USB-Kabel an den Computer an.
    Der Treiber wird automatisch installiert.
  5. Klicken Sie nach einigen Minuten auf die Schaltfläche „Start“ und dann auf „Alle Apps“.
    Öffnen Sie den Roland-Ordner. Wenn das Symbol für „Roland RD“ angezeigt wird, ist die Installation abgeschlossen.
    6. *Unter normalen Bedingungen dauert die Treiberinstallation einige Minuten.Je nach Internetkonnektivität oder bei der Installation anderer Treiber für andere Geräte kann der Vorgang bis zu zehn Minuten dauern.
    *Wenn das Roland RD verbunden ist, aber nach einer Wartezeit von zehn Minuten nicht verwendet werden kann, ist möglicherweise ein Problem aufgetreten.Siehe "Fehlerbehebung."

Ändern/Überprüfen der Treibereinstellungen

Ändern/Überprüfen der Treibereinstellungen

  1. Achten Sie darauf, vor dem Ändern der Treibereinstellungen stets alle Anwendungen zu schließen, die Roland RD verwenden. Das ist nicht erforderlich, wenn Sie die Einstellungen lediglich überprüfen möchten.
  2. Klicken Sie auf die Schaltfläche „Start“ und dann auf „Alle Apps“.
  3. Öffnen Sie den Roland-Ordner, und klicken Sie auf „Roland RD“.
  4. Das Dialogfeld "Roland RD Treibereinstellungen" wird angezeigt.
    *Sie können das Dialogfeld „Roland RD Treibereinstellungen“ auch über die Windows-Systemsteuerung öffnen.Klicken Sie in der Systemsteuerung auf „Anzeige:“, und legen Sie fest, dass Symbole angezeigt werden. Doppelklicken Sie dann auf das Symbol für Roland RD.
  5. Zum Ändern der Einstellungen, stellen Sie die gewünschten Werte ein und klicken auf [OK]. Wenn Sie die Einstellungen lediglich prüfen möchten, sollten Sie nach dem Anzeigen auf [Abbrechen] klicken.
    *Nähere Hinweise zu diesen Einstellungen finden Sie unter "Informationen zu den Optionen im Dialogfeld „Treibereinstellungen“."

Kontrollkästchen "Leichte Last"

Eine Neuinstallation vornehmen

  1. Löschen Sie den Treiber wie unter "Deinstallieren beschrieben."
    *Bei einer Deinstallation wird das Dialogfeld Treibereinstellungen initialisiert. Nähere Hinweise zu diesen Einstellungen finden Sie unter "Informationen zu den Optionen im Dialogfeld „Treibereinstellungen“."
  2. Installieren Sie den Treiber wie unter "Installation beschrieben."

Deinstallieren

  1. Trennen Sie alle USB-Geräte außer Tastatur und Maus, und starten Sie dann Windows.
    *Stellen Sie sicher, dass Sie auch das USB-Kabel des Roland RD entfernen.
  2. Beenden Sie sämtliche Anwendungen, bevor Sie mit der Deinstallation beginnen.
    *Bitte warten Sie ca. 10 Sekunden, bis diese Anwendungen vollständig beendet wurden.
  3. Klicken Sie mit der rechten Maustaste auf die Schaltfläche „Start“ und dann auf „Programme und Features“.
  4. Wählen Sie in der Liste den "Roland RD-Treiber" und klicken Sie auf "Deinstallieren/ändern."
    *Der Treiber ist nicht installiert, wenn der Roland RD-Treiber nicht in der Liste angezeigt wird.Sie müssen den Treiber nicht deinstallieren.
  5. Wenn das Dialogfeld zur Benutzerkontensteuerung angezeigt wird, klicken Sie auf [Ja].
    *Wenn Sie aufgefordert werden, ein Kennwort für ein Administratorkonto einzugeben, melden Sie sich bei Windows als Benutzer mit dem Kontotyp Administratoran, und führen anschließend die Deinstallation durch.
  6. Auf dem Bildschirm erscheint die Meldung "Der installierte Roland RD-Treiber wird deinstalliert." Klicken Sie auf [OK].
  7. Wenn die Meldung "Die Deinstallation ist abgeschlossen." angezeigt wird, klicken Sie auf [OK], um Windows neu zu starten.

Fehlerbehebung

Treiber kann nicht installiert werden

Überprüfung des Gerätestatus
Unter normalen Bedingungen dauert die Treiberinstallation einige Minuten.
Wenn das Roland RD verbunden ist, aber nach einer Wartezeit von zehn Minuten nicht verwendet werden kann, führen Sie die nachfolgenden Schritte aus, und überprüfen Sie den Status des Geräts.
  1. Schalten Sie das Roland RD ein, und schließen Sie das Gerät mit einem USB-Kabel an den Computer an.
  2. Klicken Sie mit der rechten Maustaste auf die Schaltfläche „Start“ und dann auf „Geräte-Manager“.
  3. Klicken Sie im Menü „Ansicht“ auf „Geräte nach Typ“.
  4. Klicken Sie auf „Audio-, Video- und Gamecontroller“ oder „Andere Geräte“, um das Roland RD-Gerät anzuzeigen.
  5. Wenn das Roland RD-Gerät angezeigt wird, doppelklicken Sie auf das Gerät, um den Bildschirm mit den Eigenschaften für das Roland RD anzuzeigen.
  6. Überprüfen Sie auf der Registerkarte „Allgemein“ das Feld „Gerätestatus“.
Wenn die Nachricht in dem Feld „Das Gerät funktioniert einwandfrei.“ lautet, ist die Treiberinstallation abgeschlossen.
Andernfalls ist die Treiberinstallation möglicherweise fehlgeschlagen.
Weitere Informationen zu Problemen bei der Treiberinstallation finden Sie im Abschnitt zur Fehlerbehebung.
 
Ist Ihr Computer mit dem Internet verbunden?
Für die Installation dieses Treibers muss Ihr PC mit dem Internet verbunden sein.
Stellen Sie sicher, dass Ihr PC mit dem Internet verbunden ist (beispielsweise durch die Verwendung eines Web-Browsers), und installieren Sie den Treiber erneut, indem Sie die im Abschnitt „Installation“ aufgeführten Schritte ausführen.
 
Erneute Verbindung des Roland RD
Das Gerät wurde möglicherweise nicht erkannt.
Trennen Sie das USB-Kabel des Roland RD vom PC, schalten Sie das Roland RD aus und dann wieder an, und schließen Sie das Gerät erneut an den PC an.
 
Manuelle Installation des Treibers
Je nach Internetkonnektivität wird der Treiber möglicherweise nicht automatisch installiert.
Ist dies der Fall, befolgen Sie die nachfolgenden Schritte, um den Treiber zu installieren.
  1. Schalten Sie das Roland RD ein, und schließen Sie das Gerät mit einem USB-Kabel an den Computer an.
  2. Klicken Sie mit der rechten Maustaste auf die Schaltfläche „Start“ und dann auf „Geräte-Manager“.
  3. Klicken Sie im Menü „Ansicht“ auf „Geräte nach Typ“.
  4. Klicken Sie auf „Audio-, Video- und Gamecontroller“ oder „Andere Geräte“, um das Roland RD-Gerät anzuzeigen.
  5. Wenn das Roland RD-Gerät angezeigt wird, klicken Sie mit der rechten Maustaste auf das Gerät, und wählen Sie „Treibersoftware aktualisieren“ aus.
  6. Wählen Sie „Automatisch nach aktueller Treibersoftware suchen“ aus.
    Ihr PC startet die Suche nach dem Treiber, und der Treiber wird automatisch installiert.
Die Installation ist abgeschlossen, wenn die Nachricht „Die Treibersoftware wurde erfolgreich aktualisiert.“ angezeigt wird. Klicken Sie auf „Schließen“.
Wenn das Dialogfeld "Geänderte Systemeinstellungen" angezeigt wird, klicken Sie auf [Ja], um Windows neu zu starten.
 

Auswahl/Verwendung des Roland RD-Geräts nicht möglich

Wurde der Treiber korrekt installiert?
Der Treiber wurde möglicherweise nicht ordnungsgemäß installiert.Weitere Informationen finden Sie im Abschnitt „Treiber kann nicht installiert werden“.
Wenn Sie das Roland RD nach der Installation des Treibers nicht verwenden können, installieren Sie den Treiber erneut, indem Sie die im Abschnitt „Eine Neuinstallation vornehmen“ aufgeführten Schritte durchführen.
 
Wird das Roland RD von einem anderen Programm verwendet?
Beenden Sie alle Anwendungen, die das Roland RD verwenden, und das Gerät aus- und wieder einschalten.
Wenn das Problem dadurch nicht gelöst wird, verwenden Sie das unter "Eine Neuinstallation vornehmen" beschriebene Verfahren zur Neuinstallation des Treibers.
 
Wird der Roland RD MIDI-Gerätename angezeigt?
Wenn der Gerätename nicht angezeigt wird, sollten Sie alle Anwendungen beenden, die das Roland RD verwenden, und das Gerät aus- und wieder einschalten.
Wenn das Problem dadurch nicht gelöst wird, verwenden Sie das unter "Eine Neuinstallation vornehmen" beschriebene Verfahren zur Neuinstallation des Treibers.
 
Ist der Computer in einen Energie sparen-Modus übergegangen, während das Roland RD verbunden und eingeschaltet war?
Wenn das Roland RD nach der Rückkehr aus dem Energie sparen-Modus nicht korrekt funktioniert, sollten Sie alle Anwendungen beenden, die das Roland RD verwenden, und anschließend das Roland RD aus- und wieder einschalten.
 
Wurde das USB-Kabel entfernt und wieder angeschlossen, während das Roland RD verwendet wurde, oder wurde das Roland RD aus und wieder eingeschaltet?
Wenn das USB-Kabel während der Verwendung des Roland RD entfernt und wieder angeschlossen wird oder wenn das Roland RD aus- und wieder eingeschaltet wird, kann es ggf. nicht mehr korrekt funktionieren. In diesem Fall sollten Sie alle Anwendungen beenden, die das Roland RD verwenden, und das Roland RD aus- und wieder einschalten.
 
Wenn der Computer gestartet wird, während das Roland RD noch im eingeschalteten Zustand mit dem Computer verbunden ist, wird der Treiber möglicherweise nicht richtig geladen, und Sie können das Roland RD möglicherweise nicht verwenden.
Schließen Sie in diesem Fall das Roland RD an einen anderen USB-Anschluss an, oder schalten Sie dasRoland RD nach dem Starten des Computers ein.
 

Am Anfang des Gerätenamens wird jetzt eine Zahl angezeigt (z. B. "2-").

Haben Sie vielleicht das Roland RD mit einem anderen USB-Anschluss verbunden?
Wenn Sie das Roland RD mit einem anderen USB-Anschluss verbunden haben, fügt das Betriebssystem möglicherweise am Anfang des Gerätenamens automatisch eine Zahl ein.
Dadurch sind jedoch keinerlei Problem mit dem Betrieb des Roland RD verbunden.
*Wenn Sie den Gerätenamen ohne diese Zahl wiederherstellen möchten, verbinden Sie das Roland RD mit demselben USB-Anschluss wie bei der Installation, oder installieren Sie den Treiber nach den Anweisungen unter "Eine Neuinstallation vornehmen neu."

Es wird kein Ton ausgegeben, die Wiedergabe oder Aufnahme ist nicht möglich

Wurde der Treiber korrekt installiert?
 
Wird das Roland RD von einem anderen Programm verwendet?
 
Haben Sie die unter "Vorbereitung" beschriebenen Einstellungen für die Verwendung des Roland RD-Treibers vorgenommen?
Das Roland RD kann verwendet werden, wenn diese Einstellungen vorgenommen wurden. Lesen Sie hierzu die entsprechenden Abschnitte unter "Vorbereitung."
 
Kann es sein, dass Sie den Computer während der Wiedergabe oder der Aufnahme in den Energie sparen-Modus versetzt haben?
Wenn das Roland RD nach der Rückkehr aus dem Energie sparen-Modus nicht korrekt funktioniert, sollten Sie alle Anwendungen beenden, die das Roland RD verwenden, und anschließend das Roland RD aus- und wieder einschalten.
 
Haben Sie während der Wiedergabe oder Aufnahme das Roland RD ein- oder ausgeschaltet oder das USB-Kabel entfernt und wieder angeschlossen?
Wenn Sie während der Wiedergabe oder Aufnahme das Roland RD ein- oder ausschalten oder das USB-Kabel entfernen und wieder anschließen, ist eine Aufnahme oder Wiedergabe mit dem Roland RD nicht mehr möglich. Halten Sie die Wiedergabe bzw. Aufnahme an, und versuchen Sie es erneut. Ist die Wiedergabe oder Aufnahme immer noch nicht möglich, beenden Sie alle Programme, die das Roland RD verwenden, und schalten Sie anschließend das Roland RD aus und wieder ein.
 

Ein Teil der Wiedergabe ist fehlerhaft: z. B. Noten werden vorzeitig abgebrochen oder gehen verloren

Die Probleme können möglicherweise durch die unten aufgeführten Einstellungen behoben werden.

Aktualisieren Sie den Treiber Ihrer Grafikkarte
Klick- und Poppgeräusche lassen sich evtl. durch eine Aktualisierung Ihrer Grafikkarte unterbinden.
 
Windows Update / Microsoft Update
Das Problem kann möglicherweise behoben werden, indem Sie Ihr System mit "Windows Update" oder "Microsoft Update" von Microsoft aktualisieren.
 
Überprüfen Sie die Einstellung der Energieoptionen
  1. Öffnen Sie die "Systemsteuerung," klicken Sie auf "System und Sicherheit," und anschließend auf "Energieoptionen."
    *Wenn die Symboldarstellung für Systemsteuerung ausgewählt ist, klicken Sie auf "Energieoptionen."
  2. Wählen Sie auf der Seite "Auswählen oder Anpassen eines Energiesparplans" den Energieplan "Höchstleistung."
    *Falls der "Höchstleistung"-Plan nicht angezeigt wird, klicken Sie auf "Weitere Energiesparpläne einblenden."
  3. Klicken Sie im Aufgabenbereich Höchstleistung auf "Energiesparplaneinstellungen ändern."
  4. Klicken Sie auf "Erweiterte Energieeinstellungen ändern."
  5. Klicken Sie unter Erweiterte Einstellungen auf das Symbol „+“ für "Festplatte" sowie auf das Symbol „+“ für "Festplatte ausschalten nach," um diese zu öffnen.
  6. Klicken Sie auf "Einstellung (Minuten)" und wählen Sie mithilfe des Nach-unten-Pfeils die Option "Nie" aus.
  7. Klicken Sie auf [OK], um das Dialogfeld Energieoptionen zu schließen.
  8. Schließen Sie das Fenster "Energiesparplaneinstellungen bearbeiten."
"Leistungsoptionen"-Einstellungen der System-Eigenschaften
  1. Öffnen Sie die "Systemsteuerung," klicken Sie auf "System und Sicherheit," und anschließend auf "System."
    *Wenn die Symboldarstellung für Systemsteuerung ausgewählt ist, klicken Sie auf "System."
  2. Klicken Sie links auf "Erweiterte Systemeinstellungen."
  3. Wenn das Dialogfeld zur Benutzerkontensteuerung angezeigt wird, klicken Sie auf [Ja].
    *Wenn Sie aufgefordert werden, ein Kennwort für ein Administratorkonto einzugeben, melden Sie sich bei Windows als Benutzer mit dem Kontotyp Administratoran, und nehmen Sie anschließend die Einstellungen erneut vor.
  4. Klicken Sie im Abschnitt Leistung auf [Einstellungen] und anschließend auf die Registerkarte [Erweitert].
  5. Wählen Sie "Hintergrunddienste" aus, und klicken Sie auf [OK].
  6. Klicken Sie auf [OK], um "Systemeigenschaften" zu schließen.
Derartige Probleme können gelöst werden, indem Sie im Dialogfeld Treibereinstellungen das Kontrollkästchen "Leichte Last" aktivieren.
Wenn Sie die Einstellungen ändern oder überprüfen möchten, sollten Sie hierzu das unter "Ändern/Überprüfen der Treibereinstellungen beschriebene Verfahren verwenden."
*Die geänderte Einstellung wird erst wirksam, nachdem Sie das Roland RD ein- und ausgeschaltet haben. Wenn keine Probleme auftreten, sollten Sie die Einstellung deaktiviert lassen (Standard).
