Pilote pour Windows 10

Votre ordinateur télécharge automatiquement le pilote Windows® 10 pour l'appareil Roland RD depuis Internet lorsque l'appareil Roland RD est connecté à l'ordinateur.
*Si vous utilisez un ordinateur compatible avec la fonction tactile, remplacez les mots "cliquez" et "faites un clic droit" dans ce document par "appuyez" et "exercez une pression prolongée", respectivement.

Précautions et restrictions d'utilisation

  • Des coupures de son ou d'autres phénomènes indésirables pourront se produire si les performances et les spécifications de votre ordinateur sont inadéquates. Les types et les modes d’utilisation des applications que vous exploitez ont également une incidence sur les résultats obtenus.
  • Si l'ordinateur est démarré alors que le Roland RD est encore connecté à l'ordinateur et sous tension, le pilote peut ne pas être chargé correctement et il se peut que vous ne soyez pas en mesure d'utiliser le Roland RD. Dans ce cas, connectez le Roland RD à un autre port USB ou mettez le Roland RD sous tension après le démarrage de l'ordinateur.
  • Si le Roland RD ne fonctionne pas correctement lorsque l'ordinateur passe du mode "Veille" au mode de fonctionnement normal, fermez toutes les applications utilisant le Roland RD et mettez le Roland RD hors tension, puis sous tension à nouveau.
  • Si vous débranchez le câble USB puis rebranchez celui-ci alors que le Roland RD est en cours d’utilisation, ou si le Roland RD est mis hors tension, puis sous tension à nouveau, il est possible que l'appareil cesse de fonctionner normalement. Dans ce cas, quittez toutes les applications utilisant le Roland RD et mettez le Roland RD hors tension, puis sous tension à nouveau.
  • Si votre ordinateur est soumis à une lourde charge de traitement, comme par exemple durant l’accès au lecteur CD-ROM ou au réseau alors que le Roland RD est en cours d’utilisation, le Roland RD peut cesser de fonctionner normalement. Dans ce cas, arrêtez dans un premier temps la lecture ou l’enregistrement, puis lancez à nouveau la lecture ou l’enregistrement. Si vous êtes toujours dans l’incapacité de démarrer la lecture ou l’enregistrement, fermez tous les programmes utilisant le Roland RD, mettez le Roland RD hors tension, puis à nouveau sous tension.

Installation

*Pour réinstaller ou mettre à jour le pilote, reportez-vous au chapitre "Réinstaller le pilote".
  1. Débranchez tous les périphériques USB à l'exception du clavier et de la souris, puis démarrez Windows.
  2. Vérifiez que l'ordinateur est connecté à Internet.
  3. Sous Windows, fermez l'écran des paramètres "Son" du Panneau de configuration et toutes les applications en cours d'exécution.
  4. Mettez l'appareil Roland RD sous tension et raccordez-le à l'ordinateur au moyen d'un câble USB.
    Le pilote sera automatiquement installé.
  5. Après quelques minutes, cliquez sur le bouton Démarrer puis cliquez sur "Toutes les applications".
    Ouvrez le dossier Roland. Si l'icône "Roland RD" s'affiche, l'installation est terminée.
    6. *En règle générale, l'installation du pilote s'effectue en quelques minutes.Elle peut nécessiter 10 minutes environ selon la connectivité Internet ou dans le cas où d'autres pilotes sont en cours d'installation pour d'autres périphériques.
    *Il se peut qu'un problème se soit produit si l'appareil Roland RD est connecté mais ne peut pas être utilisé après un délai de 10 minutes.Voir "Dysfonctionnements".

Modification/vérification des paramètres du pilote

Modification/vérification des paramètres du pilote

  1. Pour changer les réglages du pilote, assurez-vous d’abord de quitter toutes les applications utilisant le Roland RD. Ceci est inutile lors de la vérification des réglages.
  2. Cliquez sur le bouton Démarrer puis sur "Toutes les applications".
  3. Ouvrez le dossier Roland puis cliquez sur "Roland RD".
  4. La boîte de dialogue "Roland RD Paramètres du pilote" s’ouvre.
    *Vous pouvez également ouvrir la boîte de dialogue Roland RD Paramètres du pilote depuis le Panneau de configuration Windows.Sur le Panneau de configuration, cliquez sur Afficher par: et passez à l'affichage d'icônes, puis double-cliquez sur l'icône Roland RD.
  5. Pour modifier les réglages, effectuez les changements souhaités, puis cliquez sur [OK]. Pour simplement vérifier les réglages, assurez-vous de cliquer sur [Annuler] après avoir affiché les réglages.
    *Pour plus d’informations sur les différents paramètres, reportez-vous au paragraphe "À propos des éléments de la boîte de dialogue Paramètres du pilote".

À propos des éléments de la boîte de dialogue Paramètres du pilote

Option "Charge légère"

*La modification du réglage prend effet après avoir mis le Roland RD hors tension, puis sous tension à nouveau.

Ce bouton permet d'afficher le fichier README (ce document).

Réinstaller le pilote

  1. Supprimez le pilote tel qu’indiqué au chapitre "Désinstallation du pilote".
    *Lors de la désinstallation, les réglages de la boîte de dialogue Paramètres du pilote sont initialisés. Pour plus d’informations sur les différents paramètres, reportez-vous au paragraphe "À propos des éléments de la boîte de dialogue Paramètres du pilote".
  2. Installez le pilote tel qu’indiqué au chapitre "Installation".

Désinstallation du pilote

  1. Débranchez tous les périphériques USB à l'exception du clavier et de la souris, puis démarrez Windows.
    *Veillez également à retirer le câble USB de l'appareil Roland RD.
  2. Fermez toutes les applications avant d’effectuer la désinstallation.
    *Patientez environ 10 secondes jusqu'à ce que ces applications soient complètement fermées.
  3. Faites un clic droit sur le bouton Démarrer puis cliquez sur "Programmes et fonctionnalités".
  4. Dans la liste, choisissez le "Pilote Roland RD", puis cliquez sur "Désinstaller/Modifier".
    *Le pilote n'est pas installé si Pilote Roland RD n'apparaît pas dans la liste.Vous n'avez pas besoin de désinstaller le pilote.
  5. Si une boîte de dialogue relative à Contrôle de compte d'utilisateur s’ouvre, cliquez sur [Oui].
    *Si vous êtes invité à entrer le mot de passe d’un compte administrateur, connectez-vous à Windows en tant qu’Administrateur, puis lancez la désinstallation.
  6. L'écran affiche "Le pilote Roland RD installé va être désinstallé". Cliquez sur [OK].
  7. Lorsque le message "La désinstallation est terminée." s’affiche, cliquez sur [OK] pour redémarrer Windows.

Dysfonctionnements

Impossible d'installer le pilote

Vérifiez l'état de l'appareil
En règle générale, l'installation du pilote s'effectue en quelques minutes.
Si l'appareil Roland RD est connecté mais ne peut pas être utilisé après un délai de 10 minutes, suivez la procédure ci-dessous et vérifiez l'état de l'appareil.
  1. Mettez l'appareil Roland RD sous tension et raccordez-le à l'ordinateur au moyen d'un câble USB.
  2. Faites un clic droit sur le bouton Démarrer puis cliquez sur "Gestionnaire de périphériques".
  3. Dans le menu Affichage, cliquez sur "Périphériques par type".
  4. Cliquez sur "Contrôleurs audio, vidéo et jeu" ou "Autres périphériques" pour afficher l'appareil Roland RD.
  5. Lorsque l'appareil Roland RD apparaît, double-cliquez dessus pour ouvrir l'écran "Propriétés de l'appareil Roland RD".
  6. Sous l'onglet Général, vérifiez le champ "État du périphérique".
Si le message est "Ce périphérique fonctionne correctement." s'affiche dans ce champ, l'installation du pilote est terminée.
Sinon, il est possible que l'installation du pilote ait échoué.
Reportez-vous à d'autres rubriques dans Dysfonctionnements pour des informations sur les problèmes d'installation du pilote.
 
Votre ordinateur est-il connecté à Internet ?
Pour que vous puissiez installer ce pilote, votre ordinateur doit être connecté à Internet.
Vérifiez que votre ordinateur est connecté à Internet (par exemple en utilisant -*un navigateur Web), puis réinstallez le pilote suivant la procédure décrite dans la section "Installation".
 
Reconnectez l'appareil Roland RD
Il est possible que l'identification de l'appareil ait échoué.
Débranchez le câble USB de l'appareil Roland RD, mettez l'appareil Roland RD hors tension puis à nouveau sous tension, puis reconnectez-le à votre ordinateur.
 
Installation manuelle du pilote
Selon la connectivité Internet, l'installation automatique du pilote peut ne pas fonctionner.
Dans ce cas, suivez la procédure ci-dessous pour installer le pilote.
  1. Mettez l'appareil Roland RD sous tension et raccordez-le à l'ordinateur au moyen d'un câble USB.
  2. Faites un clic droit sur le bouton Démarrer puis cliquez sur "Gestionnaire de périphériques".
  3. Dans le menu Affichage, cliquez sur "Périphériques par type".
  4. Cliquez sur "Contrôleurs audio, vidéo et jeu" ou "Autres périphériques" pour afficher l'appareil Roland RD.
  5. Lorsque l'appareil Roland RD apparaît, faites un clic droit dessus, puis sélectionnez "Mettre à jour le pilote".
  6. Sélectionnez "Rechercher automatiquement un pilote mis à jour".
    Votre ordinateur se met à rechercher le pilote, et le pilote est installé automatiquement.
L'installation est terminée si le message "Windows a mis à jour votre pilote." s'affiche. Cliquez sur "Fermer".
Si la boîte de dialogue "Modification des paramètres système" s’affiche, cliquez sur [Oui] pour redémarrer Windows.
 

Impossible de sélectionner ou d’utiliser le périphérique Roland RD

Le pilote a-t-il été installé correctement ?
Il est possible que le pilote n'ait pas été correctement installé.Voir la section "Impossible d'installer le pilote".
Si vous ne pouvez pas utiliser le Roland RD après l'installation du pilote, réinstallez le pilote en suivant la procédure décrite dans la section "Réinstaller le pilote".
 
Le Roland RD est-il utilisé par un autre logiciel ?
Fermez toutes les applications utilisant le Roland RD et mettez le Roland RD hors tension, puis sous tension à nouveau.
Si ceci ne résout pas le problème, suivez la procédure décrite au chapitre "Réinstaller le pilote" afin de réinstaller le pilote.
 
Le nom du périphérique MIDI du Roland RD est-il affiché ?
Si le nom du périphérique ne s’affiche pas, fermez toutes les applications utilisant le Roland RD et mettez le Roland RD hors tension, puis sous tension à nouveau.
Si ceci ne résout pas le problème, suivez la procédure décrite au chapitre "Réinstaller le pilote" afin de réinstaller le pilote.
 
L'ordinateur est-il passé en mode "Veille" alors que le Roland RD est connecté et sous tension ?
Si le Roland RD ne fonctionne pas correctement lorsque l'ordinateur passe du mode "Veille" au mode de fonctionnement normal, fermez toutes les applications utilisant le Roland RD et mettez le Roland RD hors tension, puis sous tension à nouveau.
 
Le câble USB a-t-il été débranché, puis rebranché alors que le Roland RD était en cours d’utilisation ou le Roland RD a-t-il été mis hors tension, puis sous tension à nouveau ?
Si vous débranchez le câble USB, puis rebranchez celui-ci alors que le Roland RD est en cours d’utilisation, ou si le Roland RD est mis hors tension, puis sous tension à nouveau, il risque de ne plus fonctionner normalement. Si ceci se produit, fermez toutes les applications utilisant le Roland RD et mettez le Roland RD hors tension, puis sous tension à nouveau.
 
Si l'ordinateur est démarré alors que le Roland RD est encore connecté à l'ordinateur et sous tension, le pilote peut ne pas être chargé correctement et il se peut que vous ne soyez pas en mesure d'utiliser le Roland RD.
Dans ce cas, connectez le Roland RD à un autre port USB ou mettez le Roland RD sous tension après le démarrage de l'ordinateur.
 

Un chiffre tel que "2-" a été ajouté au début du nom du périphérique

Avez-vous connecté le Roland RD à un autre port USB ?
Si vous connectez le Roland RD à un autre port USB, le système d’exploitation peut ajouter automatiquement un numéro au début du nom du périphérique.
Cela n’entraîne aucun problème de fonctionnement du Roland RD.
*Pour rétablir le nom d’origine du périphérique et lui ôter ce numéro ajouté, connectez le Roland RD au même port USB que vous avez utilisé lors de sa première installation, ou réinstallez le pilote tel qu’indiqué sous "Réinstaller le pilote".

Aucun son ne sort. Impossible de lire ou d’enregistrer

Le pilote a-t-il été installé correctement ?
 
Le Roland RD est-il utilisé par un autre logiciel ?
 
Avez-vous effectué les réglages indiqués au chapitre "Avant de commencer" de façon à ce que le pilote Roland RD puisse être utilisé dans de bonnes conditions ?
Vous ne serez pas en mesure d’utiliser le Roland RD dans de bonnes conditions à moins de procéder à ces réglages. Reportez-vous aux différentes sections du chapitre "Avant de commencer".
 
Êtes-vous passé en mode "Veille" en cours de lecture ou d'enregistrement ?
Si le Roland RD ne fonctionne pas correctement lorsque l'ordinateur passe du mode "Veille" au mode de fonctionnement normal, fermez toutes les applications utilisant le Roland RD et mettez le Roland RD hors tension, puis sous tension à nouveau.
 
Durant la lecture ou l’enregistrement, avez-vous mis le Roland RD sous tension, puis hors tension ou avez-vous débranché, puis rebranché le câble USB ?
Si vous déconnectez et reconnectez le câble USB en cours de lecture ou d’enregistrement, le Roland RD ne sera pas capable de reprendre la lecture ou l'enregistrement. Arrêtez la lecture ou l’enregistrement, puis relancez l’une ou l’autre de ces opérations. Si vous êtes toujours dans l’incapacité de reprendre la lecture ou l’enregistrement, fermez tous les programmes utilisant le Roland RD, puis déconnectez et reconnectez le câble USB du Roland RD.
 

La lecture est en partie incorrecte : par ex., les notes s’interrompent prématurément ou certaines notes se perdent.

Il est possible de corriger ce type de problème en réglant les paramètres indiqués ci-dessous.

Essayez de mettre à jour le pilote de votre carte graphique
Il est possible d'éliminer les clics et autres bruits en mettant à jour le pilote de la carte graphique installée sur votre ordinateur.
 
Windows Update / Microsoft Update
Vous pouvez résoudre le problème en mettant à jour votre système grâce aux « Windows Update » ou « Microsoft Update » fournies par Microsoft.
 
Vérifiez les paramètres de gestion de l'alimentation
  1. Ouvrez "Panneau de configuration" et cliquez sur "Système et sécurité", puis sur "Options d'alimentation".
    *Si l'affichage par icônes est sélectionné dans le Panneau de configuration, cliquez sur "Options d'alimentation."
  2. Dans "Choisir ou personnaliser un mode de gestion de l’alimentation", sélectionnez "Performances élevées".
    *Si le mode "Performances élevées" n'apparaît pas, cliquez sur "Afficher les modes supplémentaires."
  3. Dans la zone Performances élevées, cliquez sur "Modifier les paramètres du mode".
  4. Cliquez sur "Modifier les paramètres d’alimentation avancés".
  5. Dans les Paramètres avancés, cliquez sur le symbole « + » de "Disque dur", puis sur le symbole « + » de "Arrêter le disque dur après" pour ouvrir l'arborescence.
  6. Cliquez sur "Paramètres (Minutes)", puis cliquez sur la flèche vers le bas pour sélectionner le paramètre "Jamais".
  7. Cliquez sur [OK] pour fermer la fenêtre Options d'alimentation.
  8. Fermez le panneau "Modifier les paramètres du mode de gestion de l’alimentation".
Réglage des "Options de performances" des propriétés de Système
  1. Ouvrez "Panneau de configuration" et cliquez sur "Système et sécurité", puis sur "Système".
    *Si l'affichage par icônes est sélectionné dans le Panneau de configuration, cliquez sur "Système."
  2. Sur la gauche, cliquez sur "Paramètres système avancés."
  3. Si une boîte de dialogue relative à Contrôle de compte d'utilisateur s’ouvre, cliquez sur [Oui].
    *Si vous êtes invité à entrer le mot de passe d’un compte administrateur, connectez-vous à Windows en tant qu'Administrateur.
  4. Cliquez d’abord sur [Paramètres] dans la section Performances, puis sur l’onglet [Avancé].
  5. Sélectionnez "Les services d’arrière-plan" et cliquez sur [OK].
  6. Cliquez sur [OK] pour fermer "Propriétés système".
Activez l'option "Charge légère" de la boîte de dialogue Paramètres du pilote.
Pour modifier ou vérifier les réglages, suivez la procédure décrite au chapitre "Modification/vérification des paramètres du pilote".
*La modification du réglage prend effet après avoir mis le Roland RD hors tension, puis sous tension à nouveau. À moins de rencontrer des problèmes, utilisez le réglage (initial) non coché.
